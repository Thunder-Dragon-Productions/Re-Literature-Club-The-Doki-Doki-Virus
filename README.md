<p><i><b>Re:Literature Club! The Doki Doki Virus</i></b> is a romance-to-horror story-based fan mod and <a href="https://sta.sh/2x0ii6hq2c3">scripted fan fiction series</a> created by TDPNeji/NejiHyuga900 for Team Salvato's 2017 visual novel <i><a href="http://ddlc.moe/">Doki Doki Literature Club!</a></i> This mod serves as a non-canon fan sequel and continuation to the "special ending" of <i>Doki Doki Literature Club!</i> and thus, should be played after the player acquires the secret ending.</p>
